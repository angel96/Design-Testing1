If your project doesn´t update dependencies properly, you will have to get into the following folder:

1. Get inside the HardDrive Disk (In My PC).
2. Go to Document and Setting.
3. Inside Student´s Folder you will see a folder named by ".m2".
4. Delete that folder.
5. Update your maven project. Remember to tick the "Update force and releases".
